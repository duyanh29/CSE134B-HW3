File Compression and Uncompression using Huffman Coding Tree:

Project Overview:

Constructed a Huffman code for a given text file and encode the file using the constructed Huffman code. The header of the compressed file has the information to construct the exact same Huffman Tree later to uncompress the file

To uncompress the compressed file, construct a Huffman code by reading the header of the compressed file and decode the compressed file using the exact same Huffman Tree

The encoded file is not necessarily smaller in size than the original file but the decoded file will have the exact same size as the original file

Tools used in the project

This program was implemented in C++

DNA Application with Data Structures:

Project Overview:

Can be used to compute the distance between each pair of DNA sequences, transmission clusters (connected components)

Created a graph ADT to represent the nodes of the graph using object-oriented design and unordered map

Designed a graph ADT with below properties for a file body-contenting a list of edges representing an undirected graph with non-negative edge weights in a CSV format:

  • Returning list of nodes in the graph
  • Returning neighbors of a node
  • Returning the edge weight between two nodes

Graph functionalities:

  • Detecting shortest unweighted/weighted sequence between any two nodes
  • Finding connected components so all edges are below a given threshold

Tools used in the project

The program was implemented in Java and tested with JUnit using Eclise IDE
